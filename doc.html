SYSTEM MANUAL

Operational protocols and error resolution for the Sheer Verification Unit.

01. OPERATIONAL PROTOCOL

1. Initialize Connection

Start the bot to register your user ID in the database.

/start

2. Request Verification

Initiate the verification sequence. The system will check your credits and queue position.

/vgpt

3. Token Injection

The bot will request your ChatGPT session token. You must extract this from your browser.

  1. Log in to chatgpt.com
  2. Navigate to https://chatgpt.com/api/auth/session
  3. Copy the entire JSON text shown on screen.
  4. Paste it directly into the bot chat.
// Example Token Format
{"user":{"id":"...","name":"...","email":"..."},"expires":"...","accessToken":"eyJhbGciOiJSUzI1NiI..."}
